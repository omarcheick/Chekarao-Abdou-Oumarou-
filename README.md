# Portfolio — Oumarou Abdou Chekarao

Portfolio personnel présentant mon expertise en Communication, Marketing Digital et Design Graphique.

Site en une seule page, autonome (HTML/CSS/JS), sans dépendance externe à installer.

## Publier ce site gratuitement avec GitHub Pages

1. Crée un nouveau dépôt sur GitHub (ex. `portfolio`).
2. Mets ce fichier `index.html` (et ce `README.md`) à la racine du dépôt.
3. Sur GitHub, va dans **Settings** → **Pages**.
4. Dans **Source**, choisis la branche `main` et le dossier `/ (root)`, puis clique **Save**.
5. Après 1-2 minutes, ton site sera visible à l'adresse :
   `https://<ton-nom-utilisateur>.github.io/<nom-du-depot>/`

## À personnaliser avant publication

- **Témoignages** : actuellement fictifs (section "Témoignages"), à remplacer par de vrais avis clients.
- **Formulaire de contact** : n'envoie rien pour l'instant (pas de backend). Pour le rendre fonctionnel gratuitement, tu peux le connecter à [Formspree](https://formspree.io) ou [Web3Forms](https://web3forms.com) en quelques minutes.
- **Réseaux sociaux** : les liens Facebook / Instagram / LinkedIn pointent vers `#` — remplace-les par tes vraies URLs dans le code (`<a href="#" class="icon-btn">`).
- **CV téléchargeable** : le bouton "Télécharger le CV" ne pointe vers aucun fichier pour l'instant. Ajoute ton CV en PDF dans le dépôt (ex. `CV_Chekarao.pdf`) et mets à jour le lien du bouton correspondant.

## Structure

Un seul fichier `index.html` contenant le HTML, le CSS et le JavaScript (images incluses en base64) — aucune installation ni build requis.

La section Réalisations affiche 29 projets avec une visionneuse plein écran (clic sur une image pour l'agrandir, clic en dehors ou touche Échap pour la refermer). Le fichier fait environ 2,5 Mo à cause du nombre d'images en haute qualité — cela reste très largement dans les limites de GitHub (100 Mo par fichier).
